## Week 1（Day 1〜7）: 即効施策の実行

### 1. 価格改定 $6 → $14.99

| 項目 | 内容 |
|------|------|
| **優先度** | ★★★（最優先・即日実行） |
| **工数** | 30分 |
| **期待効果** | 件数が同じなら売上2.5倍。$6→$14.99で月間売上が即座に増加 |
| **データ根拠** | 競合最安のNAMAEYA DLが$15。Etsyデジタル中央値$16。$6はFiverr翻訳ギグ($5)と同価格帯であり、プロ書道家+AI性格分析の価値に見合わない |
| **具体的アクション** | Stripe管理画面で価格を$14.99に変更 |
| **リスク** | CVR低下の可能性 → ただし$15は市場最安水準のため影響は軽微と想定 |
| **成功指標** | 変更後7日間のCVRが変更前の70%以上を維持 |
