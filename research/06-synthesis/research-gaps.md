# リサーチ全体の批判的評価: 盲点・未検証仮説・次の調査設計

**作成日:** 2026-02-25
**評価者の立場:** Your Kanji Name 事業戦略コンサルタント
**対象:** research/ フォルダ内の全13ファイル + docs/business-roadmap.md

---

## エグゼクティブサマリー

現在のリサーチは「市場は存在するか？」には答えている。しかし「この市場で勝てるか？」に答えるための**顧客行動レベルの分析**が決定的に欠落している。

具体的には:
- **供給側（市場規模・競合・チャネル）の調査は充実**（信頼度B〜A）
- **需要側（購買プロセス・トリガー・心理的障壁・WTP）の調査がほぼゼロ**
- **全KPI目標値が [仮説] のまま**で、事業判断の定量的根拠がない
- **競合分析の範囲が狭く**、実質的に最大の競合であるChatGPT/Google翻訳の脅威分析が不十分

**最大のリスク:** $19.99への値上げという最重要意思決定が、未検証の価格データ（price-analysis.md: C評価）に基づいている。

## 【1】盲点の発見: 完全に欠落しているテーマ

### 1-1. 購買ファネル分析の完全欠如

現在のリサーチには「認知→興味→検討→購入→シェア」の各段階で何が障壁になるかの調査が**一切存在しない**。

| ファネル段階 | 知りたいこと | 現在のリサーチの状態 |
|------------|------------|-------------------|
| **認知** | ターゲットはYKNをどう発見するか？ | チャネル分析はあるが、実際の流入経路データなし |
| **興味** | サイト訪問者の何%が興味を持つか？ | 完全に欠落。直帰率・滞在時間のデータなし |
| **検討** | 「面白そう」で終わる人と購入する人の分岐点は？ | **完全に欠落。これが最大の盲点** |
| **購入** | 決済完了率は？カゴ落ち率は？ | 完全に欠落 |
| **シェア** | 購入者の何%がSNSでシェアするか？ | 「シェアしやすいデザインに」とあるが効果測定なし |
| **リピート/紹介** | 友人に紹介するか？家族分を追加購入するか？ | **完全に欠落** |

**なぜ致命的か:** チャネルやSEOに投資しても、サイト上のコンバージョンが低ければ全て無駄になる。月間100件を達成するには、逆算してどれだけの流入が必要かを計算するためにも、CVRの推定が不可欠。

### 1-2. 購買トリガーの分析不在

「漢字名を作りたい」と思う**具体的なトリガー**の分析がない。

現在分かっていること:
- 欧米人は漢字に興味がある（kanji-fascination.md）
- 漢字タトゥーの失敗が怖い（tattoo-culture.md）
- 書道体験は人気がある（competitor-analysis.md）

**分かっていないこと:**
- 「面白そう」→「$20払おう」の心理的転換点は何か？
- どのマーケティングメッセージが最も購買行動を誘発するか？
- 訪日旅行のどのタイミングで購買確率が最も高いか？（旅行前の準備期？旅行中の体験期？帰国後の記念品期？）
- 「衝動買い」か「計画的購入」か？
- 価格ではなく「信頼」が最大の障壁ではないか？（知らないサービスに$20払えるか？）

### 1-3. リピート・紹介のポテンシャル未検討

リサーチ全体が「新規顧客の獲得」に集中しており、以下が完全に欠落:

| テーマ | 重要性 | 現状 |
|--------|--------|------|
| 1人の顧客が家族・友人分を追加購入する可能性 | 高（客単価を倍増できる可能性） | 未検討 |
| ギフト購入の比率（自分用 vs 贈り物） | 高（マーケティングメッセージの設計に影響） | ペルソナDで言及のみ。定量データなし |
| 口コミ紹介率（NPS/紹介意向） | 高（CACを劇的に下げうる） | 未検討 |
| リピート購入のパターン | 中（別の漢字名を試す等） | 未検討 |

**Etsyの競合データが示唆すること:** MyJapaneseNamePlusが2,469件販売しているが、うちリピート客がどの程度いるかは不明。もしリピート率が20%あれば、新規獲得コストの計算が大きく変わる。

### 1-4. 法的・文化的リスクの体系的調査不在

tattoo-culture.mdの末尾で「文化の盗用（cultural appropriation）批判」に1段落だけ触れているが、体系的な調査がない。

| リスク領域 | 具体的な懸念 | 調査状況 |
|-----------|------------|---------|
| **文化盗用批判** | 「日本人でない人が漢字名を持つのは文化の盗用か？」 | 1段落のみ。SNSでの実際の議論は未調査 |
| **品質事故リスク** | 提供した漢字名が不適切だった場合の評判リスク。1件の「悪い漢字名」がSNSでバイラルになる可能性 | **完全に未検討** |
| **タトゥー関連の法的責任** | 顧客がYKNの漢字名をタトゥーにして問題が起きた場合 | tattoo-culture.mdで「責任範囲外を明記」とあるのみ |
| **景品表示法・消費者保護** | 「プロの書道家による手書き」等の表現の適法性 | **完全に未検討** |
| **商標リスク** | 競合サービスの名称・ブランドとの類似性 | **完全に未検討** |
| **個人情報保護** | 名前データの取り扱い（GDPR含む） | **完全に未検討** |

### 1-5. オペレーションのスケーラビリティ分析不在

revised-roadmap.mdではPhase 3で月500件、Phase 4で月2,000件を目標にしているが:

| 検討すべき項目 | 現状 |
|--------------|------|
| 書道家1人が1日に何件制作できるか | **未調査** |
| 品質管理（全件チェック）のコスト | **未調査** |
| AI生成部分のエラー率・修正頻度 | **未調査** |
| 顧客対応（問い合わせ・クレーム）の発生率 | **未調査** |
| 国際配送のコスト・トラブル率（Premium/Ultimate） | **未調査** |
| 書道家の単価と利益率 | **未調査** |

**なぜ重要か:** 月2,000件で書道家が1件30分かかるなら1,000時間/月 = 6人以上のフルタイム書道家が必要。利益率の計算なしに価格を設定するのは危険。

### 1-6. 非観光客市場の過小評価

リサーチの大前提が「訪日外国人観光客」に固定されているが、オンラインサービスである以上、市場はそれだけではない。

| 未探索の市場セグメント | 推定規模 | 特徴 |
|---------------------|---------|------|
| **日本語学習者（非訪日）** | r/LearnJapanese 80万+人 | 通年需要。訪日に依存しない |
| **アニメ・マンガファン（非訪日）** | 全世界で数千万人規模 | 低〜中単価だが巨大なボリューム |
| **武道実践者** | 柔道・空手人口は全世界2,000万+人 | 漢字名への需要が高い。帯や道着に刺繍するニーズ |
| **日本企業と取引するビジネスパーソン** | 不明 | 名刺に漢字名を入れたいニーズ |
| **在日外国人（非観光客）** | 約300万人 | 日常的に漢字名を使いたい長期在住者 |
| **ウェディング市場** | revised-roadmapでPhase 4に記載 | 高単価。もっと早く検証すべき |

**特にr/LearnJapaneseの80万+人は、訪日に関係なく通年で漢字名に興味を持つ可能性がある。** 観光客市場は季節性があるが、この市場は安定的な需要源になりうる。

### 1-7. 顧客獲得コスト（CAC）の推定不在

チャネル戦略はあるがCACの概算が一切ない。

| チャネル | 想定CAC | 検討状況 |
|---------|---------|---------|
| SEO（自然検索） | $0〜$5/件（時間投資のみ） | 未計算 |
| TikTok（オーガニック） | $0〜$2/件（時間投資のみ） | 未計算 |
| Viator（手数料型） | $4〜$5/件（20-25%手数料） | 手数料率のみ記載 |
| インフルエンサー | $5〜$50/件（変動大） | 未計算 |
| パートナーQRコード | $2〜$4/件（レベニューシェア15-25%） | レベニューシェア率のみ記載 |
| 有料広告（未検討） | $10〜$30/件（推定） | **完全に未検討** |

**有料広告（Google Ads/Meta Ads）が選択肢に入っていないのは意図的か？** $19.99の商品でCAC $10以下なら十分に利益が出る。Phase 2以降で検討すべき。

## 【3】競合分析の盲点

### 現在の競合分析の範囲

competitor-analysis.mdは信頼度A（最も検証が進んでいるファイル）だが、分析範囲が**「漢字名作成を有料で提供するサービス」**に限定されている。

### 見落とされている実質的競合

#### 3-1. 無料の漢字名変換サイト・アプリ（部分的にカバー）

| 競合 | 分析状況 | 不足点 |
|------|---------|--------|
| Kanji Tools | 名前のみ記載 | **月間トラフィック・ユーザー数が不明**。無料ツールの利用者規模が分からなければ、「無料で満足する層」のサイズが分からない |
| Japanese Name Converter (Nolan Lawson) | 名前のみ記載 | 同上。**GitHubスター数やNPMダウンロード数で利用規模が推定可能** |
| Vondy Kanji Name Generator | 名前のみ記載 | 同上。AIツールの利用規模は急速に拡大中 |
| NAMAEYA無料版 | カバー済み | 無料→有料の転換率が不明 |

**不足:** 無料ツール群の合計月間利用者数が分からない。これが数十万人なら「漢字名に興味はあるが無料で済ませる人」が市場の大多数であり、有料への転換戦略が最重要課題になる。

#### 3-2. AI翻訳ツール（深刻な分析不足）

| 競合 | 脅威度 | 現在の分析状況 |
|------|--------|-------------|
| **ChatGPT** | **極めて高い** | revised-roadmapで「リスク: 高」とだけ記載。**具体的な品質比較が一切ない** |
| **Google翻訳** | 中 | **完全に未言及** |
| **DeepL** | 低〜中 | **完全に未言及** |
| **Claude** | 中 | **完全に未言及** |

**これは深刻な盲点。** ChatGPTに「My name is Michael. Create a kanji name for me with meaningful characters」と聞くと、かなりの品質で漢字名を提案してくる。YKNの差別化が「AI性格分析+書道家の手書き」だとしても、**顧客が「ChatGPTで十分」と判断する可能性の定量評価**がなければ事業リスクを正しく評価できない。

**検証すべきこと:**
1. ChatGPT/Claude/Geminiで漢字名を10件生成し、日本語ネイティブ3名に品質評価してもらう
2. YKNの結果と品質を比較
3. 「手書き書道」という要素がなければ、AI生成との差別化はどの程度可能か

#### 3-3. 対面の書道体験（部分的にカバー）

competitor-analysis.mdで価格・レビューは充実しているが、「なぜ顧客は$40〜$57の対面体験を選ぶのか」の理由分析が弱い。

**不足:**
- 対面体験の顧客とオンラインサービスの顧客は**同じ人か、別の人か**？
- 対面体験を選ぶ最大の理由は「体験そのもの（筆を持つ、先生と話す）」であり、YKNとは**補完関係**なのか**競合関係**なのかが曖昧
- 「YKNで漢字名を選んでから、対面書道で書いてもらう」という二段階利用の可能性は？

#### 3-4. 漢字タトゥーショップ（分析不足）

tattoo-culture.mdでYorozuya、Kanji Sensei、KanjiArt等6社を列挙しているが:

**不足:**
- これらのサービスの**月間利用者数・売上規模**が不明
- タトゥー検証サービスとYKNは「タトゥー前の漢字確認」で直接競合するが、**差別化戦略が具体化されていない**
- タトゥーショップとの**提携モデル**は言及されているが、具体的なアプローチ計画がない

#### 3-5. 完全に見落とされている競合

| 競合カテゴリ | 具体例 | 脅威度 |
|------------|--------|--------|
| **中国語書道サービス** | Chinese calligraphy name services（漢字は中国語圏でも共通） | 中。顧客が「漢字=中国語」と混同する可能性 |
| **Amazon/Zazzle/Redbubble** | カスタム漢字名グッズ | 低〜中。物販で競合 |
| **Canvaテンプレート** | 日本語フォントを使った自作ツール | 低。DIY層のみ |
| **日本語学校のサービス** | 語学学校が副業的に提供 | 低。規模が小さい |

## 【4】顧客インサイトの深さ: 不十分な点

### 4-1. 「$15〜$25払う人」と「払わない人」の違い

kanji-fascination.mdは「欧米人が漢字に興味がある」ことを豊富な引用で示しているが、**「興味がある」と「お金を払う」の間のギャップ**が分析されていない。

**仮説（検証が必要）:**

| 支払う人の特徴 | 支払わない人の特徴 |
|--------------|------------------|
| 訪日旅行中で「記念品」として購入（衝動買い） | 自宅から興味本位で検索（「面白そう」で終わる） |
| 書道作品の「本物感」に価値を感じる | ChatGPTで十分と考える |
| SNSで映える成果物が欲しい | テキストの漢字名だけで満足する |
| 贈り物としての需要（相手のために） | 自分用なら無料ツールで済ます |
| タトゥー前の「保険」として | タトゥーの予定がない |

### 4-2. 「面白そう」で終わる人と「実際に購入する」人の分岐点

**現在のリサーチで分かっていないこと:**

1. **信頼の壁:** 知らないウェブサイトにクレジットカード情報を入力するハードル。YKNにはまだレビューが少ない。Etsyの4.8★やViatorの243件レビューのような社会的証明がない
2. **「見てみないと分からない」問題:** 漢字名は注文前に結果が分からない。サンプルや事例の充実度がCVRを左右するはず
3. **「今じゃなくてもいい」問題:** 漢字名は緊急性がない。どう urgency を作るか
4. **支払方法の壁:** 旅行者はStripeで支払えるか？PayPal、Apple Pay、Google Payの対応状況は？

### 4-3. 旅行中のどのタイミングで購入確率が最も高いか

| タイミング | 仮説 | 検証方法 |
|-----------|------|---------|
| **旅行前（計画段階）** | 低〜中。「日本で何をしようか」と調べている段階。SEOで捕捉可能 | Google Analyticsの流入国・時間帯分析 |
| **旅行序盤（到着1-3日目）** | 中。日本文化への興奮が最高潮。パートナー店舗QRが有効 | パートナー店舗の注文時間帯を分析 |
| **旅行中盤** | 高？ 文化体験への関心が具体化する頃 | 購入者アンケートで「旅行何日目に購入したか」を聞く |
| **旅行終盤（帰国直前）** | 中〜高。お土産需要のピーク。「買い忘れた」層 | 購入者アンケート |
| **帰国後** | 中。旅行の余韻で追加購入。ギフト需要 | 注文の地域分布を分析（日本国外からの注文比率） |

### 4-4. グループ旅行 vs 個人旅行での購買行動の違い

| 旅行形態 | 仮説 | 重要性 |
|---------|------|--------|
| **個人旅行（FIT）** | オンラインで事前調査→計画的に購入。SEO/ブログが有効 | 高 |
| **カップル旅行** | ペア購入の可能性。「二人の漢字名セット」の訴求が有効 | 高（客単価2倍） |
| **友人グループ** | 1人が購入→「面白い！」→グループ全員が購入。バイラル効果 | 高 |
| **家族旅行** | 親が子供の分も購入。家族セットの訴求 | 中 |
| **団体ツアー** | 自由時間が限られる。ガイド経由の紹介が有効 | 低 |

**これが重要な理由:** マーケティングメッセージの設計が変わる。個人旅行者には「あなただけの漢字名」、カップルには「二人のストーリーを漢字に」、グループには「友達みんなでやってみよう」。
