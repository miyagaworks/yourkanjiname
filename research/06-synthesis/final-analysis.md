## 3. チャネル優先順位

### 優先度マトリクス

| 優先度 | チャネル | 期待効果 | コスト | 推奨時期 |
|--------|---------|---------|--------|----------|
| **S（最優先）** | TikTok / Instagram Reels | バイラル拡散。「漢字名リアクション動画」は高エンゲージメント | 低（自社制作） | 即座に開始 |
| **S** | SEO + コンテンツ | 「my name in Japanese」月間2〜5万検索。オーガニック流入 | 低〜中 | 即座に開始 |
| **A（高優先）** | Viator掲載 | 体験予約プラットフォームNo.1。欧米圏に強い | 中（手数料20〜25%） | Phase 1 |
| **A** | インフルエンサー連携 | 在日外国人YouTuber経由の認知獲得 | 中（無料体験+成果報酬） | Phase 2 |
| **B（中優先）** | パートナー店舗QRコード | ウォークイン客の取り込み。ロイヤリティ10% | 低 | Phase 1（既存） |
| **B** | Reddit自然参加 | r/japantravel（200万人）での認知。宣伝ではなく価値提供 | 低（時間のみ） | 即座に開始 |
| **C（低優先）** | 旅行メディア掲載 | Lonely Planet、Japan Guide等。信頼性の付与 | 中（PR活動） | Phase 2〜3 |
| **C** | GetYourGuide / Klook | Viatorより日本での存在感が弱い | 中 | Phase 2以降 |

### チャネル戦略の根拠

1. **SNS（TikTok/Instagram）が最高効率:** 観光庁調査でSNSが旅行前情報源の1位（30〜35%）。漢字名は視覚的インパクトが高く、シェアされやすい
2. **SEOは持続的な流入源:** 「my name in Japanese」の月間検索ボリュームは2〜5万。この検索意図はYour Kanji Nameのサービスと完全に一致
3. **Viatorは欧米圏への最短経路:** 書道体験カテゴリで500+レビュー、4.8/5の事例あり。手数料は高いが集客力が圧倒的
4. **パートナー店舗は補完的:** メインチャネルではなく、オンライン集客の補完として位置づけるべき

## 4. 現ロードマップの検証

### business-roadmap.md の各施策をデータで検証

#### Phase 1: 基盤固め（現在〜3ヶ月）

| 施策 | 判定 | 理由 |
|------|------|------|
| パートナー開拓（広島→京都→東京） | **修正すべき** | パートナーは補完チャネル。**オンライン集客（SNS/SEO）を最優先に追加すべき** |
| QRコードポスター配布 | **やるべき（維持）** | 低コストで実行可能。ただし主力ではなく補助的に |
| 口コミ促進（SNSシェア設計） | **やるべき（最強化）** | データが裏付ける最重要施策。シェア用画像・動画テンプレを本格化 |
| 多言語対応強化 | **やるべき（維持）** | 欧米圏が主要ターゲットであることがデータで確認 |
| KPI: 月間100件 | **修正すべき** | **価格を$6→$15に引き上げた上で100件を目指すべき**。$6×100=$600では事業として成立しない |

#### Phase 2: 認知拡大（3〜6ヶ月）

| 施策 | 判定 | 理由 |
|------|------|------|
| インフルエンサー連携 | **やるべき（最強化）** | TikTok/Reelsの「漢字名リアクション動画」は高バイラルポテンシャル |
| 旅行メディア掲載 | **やるべき（優先度下げ）** | 効果はあるがSNS/SEOより優先度低い |
| Googleマップ連携 | **やるべき（維持）** | Google Maps利用率70〜80%。パートナー店舗のGBPに記載は有効 |
| 空港・駅広告 | **不要（Phase 2では時期尚早）** | コスト対効果が不明。Phase 4以降で検討 |
| TripAdvisor活用 | **修正すべき** | **Viator掲載をPhase 1に前倒し**。TripAdvisorよりViatorの方が直接予約に繋がる |

#### Phase 3: 商品拡充（6〜12ヶ月）

| 施策 | 判定 | 理由 |
|------|------|------|
| 書道原本郵送 +$30 | **修正すべき** | **+$39.99を推奨。** Etsy相場（原本$30〜$80）と比較して安すぎる |
| 額装オプション +$50 | **修正すべき** | **+$69.99を推奨。** Etsy額装品相場（$50〜$150）と比較して安すぎる |
| Tシャツ/トートバッグ +$25 | **やるべき（維持）** | POD連携で低リスクで実現可能 |
| 印鑑（はんこ）+$40 | **やるべき（維持）** | ユニークな商品。Etsy印鑑$15〜$40の相場と整合 |
| NFT証明書 +$10 | **不要** | NFT市場は縮小。労力対効果が低い |
| KPI: 平均客単価$15 | **修正すべき** | **$25〜$35を目指すべき。** バンドル戦略で達成可能 |

#### Phase 4: スケール（1〜2年）

| 施策 | 判定 | 理由 |
|------|------|------|
| 書道家ネットワーク構築 | **やるべき（維持）** | スケーラビリティの要。月500件超で必須 |
| 法人向けサービス | **やるべき（維持）** | ホテル・旅行会社へのB2B展開は有望 |
| ウェディング市場 | **やるべき（維持）** | ニッチだが高単価が見込める |
| 企業研修連携 | **やるべき（維持）** | 法人契約で安定収益 |
| フランチャイズ展開 | **修正すべき** | 時期尚早。**まずViator等のプラットフォーム経由で海外展開**を先に |

#### Phase 5: ブランド確立（2〜3年）

| 施策 | 判定 | 理由 |
|------|------|------|
| 旗艦店舗オープン | **修正すべき** | **ポップアップストアから開始。** 常設店舗は固定費リスクが高い |
| アプリ化 | **やるべき（維持）** | 漢字名アプリは市場に不在。先行者優位あり |
| 文化庁・観光庁連携 | **やるべき（維持）** | 公的認知はブランド信頼性に直結 |
| 海外展開（Japan Expo等） | **やるべき（維持）** | フランス・パリのJapan Expoは年30万人来場 |

## 6. 最優先アクション（今すぐ）

データに基づき、**即座に着手すべき5つのアクション:**

### 1. 価格改定: $6 → $14.99
- **根拠:** 競合の中央値$15〜$30。$6は「自動生成ツール」と混同されるリスク
- **方法:** Stripe設定の`priceUsd`を変更するだけ。コード変更は1行

### 2. TikTokアカウント開設・初投稿
- **根拠:** TikTokは旅行先選定への影響力が急上昇中（40%が影響受ける）
- **コンテンツ案:** 「外国人の名前を漢字にしてみた」リアクション動画

### 3. SEO記事3本を制作
- **根拠:** 「my name in Japanese」月間2〜5万検索
- **記事案:** "How to Get Your Name in Kanji", "Kanji Name Meaning Guide", "Japanese Calligraphy Name Experience"

### 4. Viatorに出品申請
- **根拠:** 書道体験のレビュー500+、評価4.8/5の事例あり。欧米圏の予約No.1プラットフォーム
- **カテゴリ:** 「オンライン文化体験」

### 5. SNSシェア用テンプレートの本格化
- **根拠:** 既存のシェア促進策はあるが、TikTok/Reels向けの動画テンプレートが不足
- **アクション:** 書道作品の完成→名前の意味解説を15秒動画にするテンプレート

## 7. リスクと注意点

| リスク | 可能性 | 対策 |
|--------|--------|------|
| ChatGPT等のAIで漢字名を自分で生成 | 高 | 「書道家の手書き」がAIでは代替不可能な価値。体験全体のパッケージで差別化 |
| 価格引き上げで離脱率上昇 | 中 | A/Bテストで段階的に検証。$9.99→$14.99→$19.99 |
| 漢字タトゥー市場は文化的感度の問題 | 中 | 「文化的に正しい漢字」の保証を前面に出す。攻めすぎない |
| WebSearch未使用のためデータが古い | 中 | 本レポートの数値を公式ソースで検証する回を設ける |
