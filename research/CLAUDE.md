# リサーチプロジェクト: Your Kanji Name

## 目的
欧米人観光客が漢字名にどのような心理的関心を持っているかを調査し、
データに基づいたマーケティング戦略とロードマップを構築する。

## ルール
- すべての調査結果は該当フォルダにmarkdownファイルとして保存すること
- 各ファイルにはソースURL・日付を必ず記載すること
- 推測と事実を明確に区別すること
- 定量データがある場合は必ず数字を記録すること
```

---

## Step 2: サブエージェント並列リサーチ

ここがClaude Codeの真価です。以下のように指示します。
```
サブエージェントを5つ使って、以下の調査を並列で実行して。
各結果はそれぞれのフォルダにmarkdownで保存すること。

1. research/01-market/inbound-stats.md
   → 訪日外国人の最新統計（国籍別、消費傾向、体験型消費の割合）
   JNTOの公開データ、観光庁の統計を中心に調査

2. research/02-psychology/kanji-fascination.md
   → 欧米人が漢字・日本語に興味を持つ心理的背景の調査
   Reddit r/japantravel, r/LearnJapanese, r/JapanTravel の投稿分析
   「kanji name」「kanji tattoo」関連の議論を探して

3. research/03-competitors/competitor-analysis.md
   → 類似サービスの競合分析（漢字名作成、書道体験、漢字タトゥー翻訳）
   価格帯、レビュー内容、提供形態を整理

4. research/04-channels/tourist-touchpoints.md
   → 訪日外国人が旅行前〜旅行中に使う情報源・タッチポイント調査
   どこで情報を得て、何に影響されて体験を選ぶのか

5. research/02-psychology/tattoo-culture.md
   → 漢字タトゥー文化の分析（欧米でどの程度人気か、なぜ人気か）
   この層とサービスの接点はあるか
```

各サブエージェントが独立したコンテキストで動くため、メインの会話がクリーンなまま保たれます。

---

## Step 3: 深掘り調査（メインセッション）

サブエージェントの結果が出たら、メインセッションで深掘りします。
```
research/02-psychology/kanji-fascination.md を読んで。

この結果をもとに、以下をさらに調査して：
- 欧米人が「自分の名前を漢字にする」ことに具体的にどんな価値を感じるのか
- 「意味を選べる」ことの重要性はどの程度か
- 既存の書道体験のTripAdvisorレビューから、何が満足度を決めているか調べて
- 結果は research/02-psychology/value-drivers.md に保存
```

**思考レベルの指定**も効果的です：
```
think harder about this:
research/02-psychology/ のファイルをすべて読んで、
欧米人が漢字名に惹かれる心理的動機を5つのカテゴリに整理して。
単なるまとめではなく、マーケティングメッセージに変換できるレベルの
インサイトを導き出して。結果は research/06-synthesis/psychological-insights.md に保存。
```

## Step 4: 統合分析とロードマップ再構築

すべての調査が揃ったら、統合させます。
```
research/ フォルダ内のすべてのファイルを読み込んで。

以下の観点で統合分析を行い、research/06-synthesis/final-analysis.md に保存して：

1. 市場機会の定量評価
   - ターゲット市場の実際の規模
   - 獲得可能な市場シェアの現実的な推計

2. 顧客ペルソナ（データに基づく）
   - 最も関心が高い国籍・年齢層
   - 購買に至る心理的トリガー
   - 価格感度

3. チャネル優先順位
   - どこで認知を取るのが最も効率的か
   - パートナー戦略の妥当性検証

4. 現ロードマップの検証
   - business-roadmap.md の各Phase施策をデータで検証
   - 「やるべき」「修正すべき」「不要」に分類

5. 改訂版ロードマップの提案
   - データに基づいた優先順位の再設定
```

## Step 5: カスタムコマンド化

繰り返し使うリサーチは、`.claude/commands/` にコマンドとして保存しておきます。
```
以下のカスタムスラッシュコマンドを作って：

/research-update
→ 競合の最新動向、訪日外国人統計の更新、SNSでの漢字関連トレンドを
  サブエージェントで並列調査し、research/ フォルダを更新する

/analyze-reviews
→ TripAdvisor、Google Reviewsの書道体験レビューを調査して
  顧客インサイトを更新する
